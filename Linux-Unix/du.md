# du 

ディレクトリごとに使用量を調べて表示するためのコマンド

example 
```
 user ~$ du -hs
```

some option lists
- `-h`
  - `-human-readable`
  - サイズに応じて読みやすい単位に変更する
- `-s`
  - `--summarize`
  - 指定したディレクトリの合計のみを表示する（サブディレクトリの行が表示されなくなる）
